## Other Features
### App directory
Discord is itself an "Application" or "App" on your device. Additionally, Discord offers an *app directory*, with a variety of games and features. These are detailed in this [Guide to Discord Apps](https://discord.com/blog/how-to-use-discord-apps).

### Threads
Threads allow users to continue conversations on a particular topic without getting mixed around in a big chat. This is particularly useful for scheduling meetings or arranging carpools. Here's a useful [guide to threads](https://support.discord.com/hc/en-us/articles/4403205878423-Threads-FAQ).
### Nitro & Server Boosting
Nitro is Discord's subscription service. A $10/month subscription grants two server boosts, which can be used on your club server to gain various benefits! These include higher-quality streams making online movie nights more feasible and custom emojis to build club identity. You can consult [this FAQ](https://support.discord.com/hc/en-us/articles/360028038352-Server-Boosting-FAQ) for more information on boosts.
### More Permissions 
Discord contains a multitude of permissions for every purpose. This document covers the most pertinent permissions to club leaders. However, if you wish to learn more about permissions refer to [Permissions on Discord](https://discord.com/community/permissions-on-discord-discord).
