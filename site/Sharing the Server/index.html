    Sharing the Server

    Now that you have a working server, you'll want to get your club members in! There are two methods to get club members in your server: invite links, and QR codes.

    On PC

    Evergreen Invite Link

    This is where we will teach you how to create a permanent invite link to easily share your server with others.

    1. Click on the name of your discord server. You should see these list of options:
    2. Click on the Invite People option. If you have been using Discord before, you'll see a list of friends on Discord as well as any group chats you're in. Invite People is the 2nd option in the dropdown
    3. There will also be a "Or, send a server invite link to a friend". It looks like this:
    4. Click the copy button to copy the invite link. You can now paste the link and send it off.

    QR Code

    1. Copy your invite link
    2. Paste it in the Your url field of QR Code Monkey.
    3. Click Create QR Code
    4. Click Download PNG

    On mobile

    Evergreen Invite Link

    1. Start by clicking the people button

    2. Click copy link(should be the 2nd button)

    3. Paste the link by holding your finger down, and select "Paste" (on iPhone, it should be the first option you see)

    QR Code

    1. Start by clicking the people button

    2. Click on the QR Code button. It's the third button from the left.

    3. You should now see an actual QR Code on your screen. Show this to invitees or screenshot and print!
